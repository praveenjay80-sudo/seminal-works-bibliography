# 📚 Seminal Works Bibliography Generator

A comprehensive interactive application for exploring the formal sciences taxonomy and generating AI-powered bibliographies for 1,482+ topics.

## Features

- **1,482 Items**: Complete exhaustive taxonomy across 6 formal science domains
- **AI-Powered Bibliographies**: Generate seminal works lists using GPT-4o
- **Advanced Search**: Instantly search across all items
- **Domain Filtering**: Browse by Mathematics, Logic, Computer Science, Statistics, and more
- **Dark Theme**: Beautiful, modern UI with optimized accessibility
- **Fast**: All data embedded for instant loading

## Domains

- **I. Mathematics** - Pure and applied mathematics
- **II. Logic and Foundations** - Formal logic, set theory, proof theory
- **III. Computer Science** - Theory, algorithms, AI/ML
- **IV. Statistics** - Probability theory, statistical methods
- **V. Related Formal Fields** - Physics, operations research
- **VI. Emerging and Interdisciplinary** - New hybrid fields

## Usage

1. Open `index.html` in a web browser
2. Enter your OpenRouter API key (pre-configured for deployment)
3. Search or browse domains
4. Click "Generate Bibliography" on any item
5. View AI-generated seminal works in the popup

## Tech Stack

- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **API**: OpenRouter (GPT-4o)
- **Data**: Complete formal sciences taxonomy embedded

## Deployment

Deployed on Vercel at [seminal-works-bibliography.vercel.app](https://seminal-works-bibliography.vercel.app)
